# WotStat Debug Utils

> Задумка была сделать различные Gizmos маркеры и боксы для отладки, но рендеринг окон не синхронизирован с рендерингом сцены, поэтому маркеры иногда отстают от реального положения объектов. Так что может потом.

Можно поставить мод и в игре или ангаре нажать `V` что бы поставить маркер в `0,0,0` или координаты танка.
